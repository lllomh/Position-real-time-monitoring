# 📊 前言

这是一个用于 **实时统计主流前端框架在各城市职位需求数量** 的可视化项目。

系统会定期抓取各城市相关岗位数据，并在每天 **23:59
自动保存统计记录**，逐渐积累形成历史数据。通过折线图等可视化方式，可以清晰地观察到各个前端框架在招聘市场中的
**需求变化趋势**。

该项目主要用于
**技术趋势观察和学习研究**，对开发者了解前端技术生态具有一定参考价值。

> ⚠️ **说明：**\
> 本项目仅用于技术研究与个人兴趣，不涉及招聘或商业用途。

# 🚀 项目运行

## 环境要求

由于项目使用了 **ES6 / ES7 新特性**，请确保 Node.js 版本满足以下要求：

-   Node.js **≥ 6.0**

⚠️ 建议使用稳定版本 Node.js，避免使用测试版本。

------------------------------------------------------------------------

## 克隆项目

``` bash
git clone https://github.com/lllomh/Position-real-time-monitoring.git
cd Position-real-time-monitoring
npm install
```

------------------------------------------------------------------------

## 开发环境运行

``` bash
npm run dev
```

浏览器访问：

    http://localhost:8080

------------------------------------------------------------------------

## 生产环境构建

``` bash
npm run build
```

构建完成后会生成 `dist` 文件夹，将该文件夹部署到服务器即可正常访问。
